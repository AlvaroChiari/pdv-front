# Interface de Estoque

Interface para o gerenciamento de produtos, conectada à API Java Spring Boot.


## 🚀 Tecnologias
- **HTML:** Estrutura semântica.
- **CSS:** Estilização de layout.
- **JavaScript:** Consumo de API usando `fetch`, manipulação de DOM e pesquisa em tempo real.

## 📖 Funcionalidades
- **Cadastro:** Formulário que alterna entre "Salvar" e "Atualizar".
- **Busca:** Filtro de produtos conforme o usuário digita.
- **CRUD:** Integração total com o Back-end para criar, ler, editar e excluir.

## ⚠️ Requisito Importante
Para que este Front-end funcione corretamente, a **(https://github.com/AlvaroChiari/pdv-api)** deve estar rodando localmente na porta `8080`.
